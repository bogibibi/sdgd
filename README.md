# Ну... погоди... Bot

Telegram-бот магазина с Mini App (WebApp), системой оплаты USDT и доставкой.

## Возможности

### Роли
- **Админы** (макс. 3) — полный доступ к панели управления
- **Оператор Bakery** — получает заказы на выпечку
- **Оператор Main** — получает основные заказы
- **Доставщики** (макс. 2) — принимают и выполняют доставки

### Безопасность
- Капча при первом входе (подсчёт эмодзи среди обманок)
- Бан на 24 часа после 3 неверных попыток ввода пароля админа
- Anti-spam: 2+ запроса в секунду = игнор на 3 секунды
- Rate limiting на сайте (защита от DDoS)

### Мультиязычность
5 языков: Русский, English, 한국어, 日本語, 中文  
Выбор языка после прохождения капчи.

### Доставка
- Галочка «Доставка 10 USDT» в окне заказа на сайте
- Бот спрашивает адрес доставки после выбора галочки
- Курьерам приходит уведомление с кнопкой «Принимаю заказ»
- После выполнения — кнопка «Заказ выполнен», копия всем админам

### Зарплаты

| Роль | Формула |
|------|---------|
| Оператор Bakery | 100% с выпечки + 15% с остальных позиций (без доставки) |
| Оператор Main | 85% без доставки / 80% с доставкой |
| Доставщик | 10 USDT + 5% от чека |

### Уведомления после оплаты
- Заказ с Main позициями → копия **всем** операторам
- Заказ только из Bakery → копия только оператору выпечки + админам

### Help
Клиентам показываются только операторы Bakery и Main (без доставщиков).

### Сброс статистики
Админы могут сбросить всю статистику через панель управления (требуется пароль).

## Установка

```bash
pip install -r requirements.txt
```

## Настройка

1. В `bot.py` укажите `BOT_TOKEN` и `WEBAPP_URL`
2. Скопируйте `payment_config.example.json` → `payment_config.json` и заполните API-ключи
3. При первом запуске `config.json` создастся автоматически

## Запуск

```bash
# Бот
python bot.py

# Веб-сервер (порт 8080)
python web_server.py
```

## Оплата по TX hash

После выбора сети покупатель отправляет TX hash транзакции. Бот проверяет:
- **TRC20** — через TronScan + TronGrid
- **BEP20** — через публичный BSC RPC
- **TON** — сохраняет hash для ручной проверки оператором

Файл `payment_config.json` (не коммитится):
```json
{
  "TRONGRID_API_KEY": "your_key",
  "ETHERSCAN_API_KEY": "your_key",
  "TONCENTER_API_KEY": "",
  "BSC_RPC_URL": "https://bsc-dataseed.binance.org/"
}
```

## Пополнение склада

Команда `/add` (только для админов):
1. Фото с описанием
2. Название товара (как в WebApp)
3. Вес (число)
4. Район
5. Количество

Бот фильтрует районы по весу: если покупатель выбрал 1g, покажутся только районы с товаром именно на 1g.
